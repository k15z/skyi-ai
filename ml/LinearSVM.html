    LinearSVM.coffee

    This is a ridiculously lightweight linear support vector machine implementation. It uses gradient descent to find a hyperplane (binary classification only) between the +1 and -1 samples. It may or may not converge on the optimal hyperplane - depending on performance, may be reimplemented as a kernel support vector machine.

        fit: (x, y) ->
        num_samples = x.length
        num_features = x[0].length
        if not @bias or not @weight
            @bias = 0.5 - Math.random()
            @weight = (0.5 - Math.random() for [0...num_features])
        for e in [0...@epochs]
            incorrect = 0
            for i in [0...num_samples]
                if @predict_one(x[i]) * y[i] < 1
                    incorrect++
                    for j in [0...@weight.length]
                        @weight[j] += @l_rate * x[i][j] * y[i]
                        @bias += @l_rate * y[i]
            accuracy = 1.0 - incorrect / num_samples
            if @verbose
                console.log("epoch #{e}: #{accuracy} acc")
